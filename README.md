# Kit Reto 8° Colombia

Aplicación web estática y descargable para preparación a grado octavo.

## Uso
1. Descomprima el archivo ZIP.
2. Abra `index.html` en Chrome, Edge o Firefox.
3. Resuelva los retos por área. El progreso se guarda en el navegador mediante localStorage.

## Contenido
- 7 áreas: Matemáticas, Ciencias Naturales, Ciencias Sociales, Lenguaje, Inglés/Idiomas, Física y Química.
- 20 ejercicios por área, 140 en total.
- Explicación, ejemplo guiado, problema ABP, estrategias didácticas y gamificación con XP.

## Referentes curriculares base
Diseñado con orientación de los Estándares Básicos de Competencias y Derechos Básicos de Aprendizaje del MEN/Colombia Aprende para educación básica secundaria.


## Diseño responsive
La interfaz fue ajustada para escritorio, tablet y celular. En pantallas pequeñas se activa un menú plegable de áreas, las tarjetas pasan a una sola columna, los botones ocupan el ancho disponible y los textos largos se ajustan para evitar desbordamientos horizontales.
